# Source Notes — Clean V3

This package was rebuilt from the uploaded `gear-master-overhaul(2).zip` master list. The `originalKdmName` field is retained only as an audit/source anchor. The player-facing `ourGameName` field has been regenerated with controlled substitutions to avoid generated prefix clutter and accidental user-like names.

Status wording follows the project status model: Bleed, Burn and Poison decay at end of turn; Marked, Exposed, Snared, Shock, Blind, Stagger and Guarded are combat-control states.
