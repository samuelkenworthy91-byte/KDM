# Loadout Rule Patch v8

## Weapon rule
The v7 phrase "no duplicate weapon names" is removed.

Correct rule:
- Maximum 2 equipped weapon items.
- Duplicate weapon names are allowed.
- Maximum 2 distinct weapon names.
- Example legal loadouts:
  - KATAR + KATAR
  - KATAR + BOW
  - SHIELD + SHIELD
- Example illegal loadouts:
  - KATAR + BOW + SPEAR
  - KATAR + KATAR + BOW

## Other slot rules unchanged
- Max equipment: 9.
- Armour slots: Head, Chest, Waist, Arms, Legs.
- One armour item per body slot.
- Jewellery slot remains separate.
- Utility gear fills remaining capacity.
