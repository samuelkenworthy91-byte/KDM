# KDM
# Lantern Deckbuilder

Play latest main build: https://samuelkenworthy91-byte.github.io/KDM/

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Every push to `main` builds the Vite app and deploys `dist` to GitHub Pages. The main branch is intended to be playable from the Pages link without Codespaces.
